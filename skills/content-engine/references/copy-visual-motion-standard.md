# Copy, Visual, And Motion Standard

Use this as the final creative standard for 채널 packages.

## Copy

Write from the viewer's life, not the source's feature list.

Bad:

- "OpenAI가 업무용 AI 교육 코스를 공개했습니다."
- "TTS, ASR, Realtime 기능을 제공합니다."
- "에이전트 워크플로우를 최적화합니다."

Better:

- "ChatGPT, 아직도 감으로 쓰고 있나요?"
- "녹음 다시 듣지 말고, 회의록부터 뽑아보세요."
- "AI한테 회사 계정 줄 거면, 이 가드레일부터 보세요."

## Card Structure

Default 7-card structure:

1. Hook: viewer pain or surprising result.
2. What changed: one plain Korean sentence.
3. Proof: official/source screen or demo.
4. Use case: who uses it and what becomes easier.
5. How to try: command, step, workflow, or first action.
6. Caveat: cost, license, security, quality, access, or setup note.
7. CTA: save/comment/follow with a clear reason.

## Visual Rules

- Use GmarketSans for Korean.
- Use real source screenshots, official media, demo frames, or product UI whenever possible.
- Keep the source visual recognizable. Do not over-blur, grayscale, or darken it into a texture.
- Use GPT images only when they explain the use case better than a screenshot.
- Make generated images topic-specific:
  - Codex -> coding workspace, terminal, diff, mobile approval.
  - Claude -> Claude logo/theme, chat/workbench, code/design/security context.
  - Voice tool -> waveform, script, microphone, voice content output.
  - Browser/MCP -> search page, browser side panel, extracted result.
  - Security -> token/key/permission/checkpoint metaphor with real UI cues.
- Avoid random robots, neon blobs, abstract AI brains, and decorative fireworks unless the source is playful.

## Motion Rules

Reel is required by default.

Minimum motion jobs:

- typed prompt/search/command;
- source screenshot zoom or crop pan;
- callout or checklist reveal;
- cursor/tap pulse;
- before/after or result reveal;
- CTA keyword pop.

Do not deliver a slideshow of PNG cards.

## Safe Zones

For Reels:

- keep core text away from right-side action buttons;
- keep CTA above Instagram profile/caption area;
- avoid bottom-heavy overlays;
- inspect hook, middle, and CTA frames after render.

For carousels:

- inspect full 4:5 card and centered 1:1 grid crop;
- fix number badges overlapping headlines;
- avoid chips touching titles;
- avoid lonely Korean particles or one-word orphan lines;
- keep source/footer small but readable.

## Humanize QA

Run Korean copy through a QA pass:

- Does it sound like a Korean creator would say it?
- Is it too formal?
- Is it a translated noun-chain?
- Did the cute tone weaken the factual point?
- Can a non-expert explain the card back after 2 seconds?

Keep `~욤` only where friendly and natural. Serious security, billing, legal, privacy, or data-loss topics should be warmer than corporate copy but not cutesy.
