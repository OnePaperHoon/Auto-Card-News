# Auto Card News

把一个来源链接变成韩语 AI 新闻卡片、Reel 方案、标题文案和来源整理的 Agent Skill 示例。

## 安装

Codex:

```bash
curl -fsSL https://raw.githubusercontent.com/YOUR_ORG/YOUR_REPO/master/install.sh | bash
```

Claude Code:

```bash
curl -fsSL https://raw.githubusercontent.com/YOUR_ORG/YOUR_REPO/master/install-claude.sh | bash
```

## 使用

```text
Use the content-engine skill.

Source:
<SOURCE_URL>

Create a Korean AI-news carousel, Reel plan, caption, and source links.
Use real screenshots or demos when possible.
```

## 重点

- 不只复制官方博客
- 验证来源和社区反馈
- 使用真实截图、演示或产品画面
- 写成韩国用户容易理解的短文案
- 同时规划卡片和短视频
