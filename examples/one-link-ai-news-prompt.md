# One-Link AI News Prompt

Copy this prompt after installing the skills.

## Codex

```text
$content-engine
<SOURCE_URL>

카드뉴스 자동화로 만들어줘.

목표:
- 인스타 카드뉴스 7장
- 릴스 15~20초 플랜
- 인스타 캡션
- 출처 링크
- 실제 사용 이미지/영상 후보

스타일:
- 한국 사람이 바로 이해하는 말투
- 기능 설명보다 체감 결과 중심
- 첫 장은 후킹 강하게
- 줄글 금지
- PPT 느낌 금지
- 가능하면 실제 화면, 제품 이미지, 데모 영상, GitHub 화면을 활용
- 마지막에는 저장/댓글/팔로우 유도
```

## Claude Code

```text
Use the content-engine skill.

Source:
<SOURCE_URL>

Make a Korean AI news content package for 채널:
- 7-card Instagram carousel
- Reel plan
- caption
- source links
- visual references
- QA checklist

Do not copy the source post directly.
Verify the underlying sources, explain why Korean viewers should care, and make the copy feel human.
```
